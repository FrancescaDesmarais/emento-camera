# Visual Style Guide

This is the visual style guide for Claude to follow for the Emento-Camera project.  

### Overall Aesthetics
- Simple & clean: the goal is to highlight function, not visual design
- Claude is building simple, wireframe examples of working prototypes
- Imagine Claude is ‘sketching’ with code, as if drawing initial ideas on paper

### Colors
- Monochrome: only use black, white and shades of dark green: HEX code #228B22
- Shading and tinting is allowed to create subtle depth

### Icons
- Claude can build very simple icons, no elaborate details

### Images
- Claude cannot add images to the prototype
- Francesca can suggest images in very specific situations
